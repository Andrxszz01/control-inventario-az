Requisitos

Lo necesario para instalar y trabajar sin friccion.

Version resumida para ventas e implementacion, sin repetir bloques largos de documentacion tecnica.

Equipo recomendado

Minimo

Procesador tipo Intel Core i3 o equivalente, 4 GB de RAM, 500 MB libres y pantalla de 1024 x 768.

Recomendado

Intel i5 o Ryzen 5, 8 GB de RAM, SSD y pantalla de 1280 x 720 o superior para mejor fluidez.

Sistema operativo

Windows 10 u 11 de 64 bits. Versiones antiguas no se recomiendan para operacion comercial.

Impresora

Opcional para tickets. Se recomienda impresora termica de 58 mm u 80 mm con driver Generic/Text Only u oficial.

Internet

No es obligatorio para ventas, caja e inventario. Solo ayuda para soporte remoto, respaldo o actualizaciones.

Espacio y crecimiento

La instalacion base ronda unos cientos de MB; el crecimiento depende del volumen de productos y movimientos.

Para negocio real conviene usar una PC dedicada al sistema, hacer respaldos frecuentes y controlar el acceso del usuario administrador.
