Guia de impresora

Configuracion breve para imprimir tickets.

Paso a paso resumido para dejar lista una impresora termica usando Generic / Text Only en Windows.

Pasos

1. Abrir configuracion de Windows

Entra a Configuracion y busca la seccion de impresoras y escaneres.

2. Agregar dispositivo

Conecta la impresora y usa la opcion de agregar nueva impresora.

3. Seleccion manual

Si Windows no la detecta, elige configuracion manual y selecciona el puerto correcto.

4. Driver correcto

Usa fabricante Generic y modelo Generic / Text Only. Si no aparece, ejecuta Windows Update.

5. Nombrar y establecer

Guarda la impresora con un nombre claro y colócala como predeterminada si se usará para tickets.

6. Prueba real desde el sistema

Haz una venta de prueba en Control AZ y confirma que el ticket salga correctamente.

Si el ticket sale con simbolos raros o en blanco, casi siempre el problema es el driver incorrecto o el puerto equivocado.
