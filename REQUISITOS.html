Requisitos del Sistema

Especificaciones mínimas y recomendadas para que el sistema funcione correctamente en la computadora del cliente.

Resumen rápido

1. Requisitos de hardware 2. Sistema operativo 3. Software necesario 4. Impresora (opcional) 5. Internet y red 6. Instalación 7. Espacio en disco 8. Recomendaciones

1. Requisitos de hardware

Mínimos

  • Procesador: Intel Core i3 o equivalente
  • RAM: 4 GB
  • Disco libre: 500 MB
  • Pantalla: 1024 x 768

Recomendados

  • Procesador: Intel i5 / Ryzen 5 o superior
  • RAM: 8 GB o más
  • Disco: SSD con al menos 2 GB libres
  • Pantalla: 1280 x 720 o superior
Entre más productos y ventas registre el negocio, mayor rendimiento tendrá con 8 GB de RAM y disco SSD.

3. Software necesario

El instalador incluye lo necesario para operar. El cliente no debe instalar Node.js ni base de datos manualmente.

ElementoNecesarioIncluido
Aplicación de escritorioSíSí (.exe)
Driver de impresoraSolo si imprime ticketNo
Navegador webPara manuales HTMLGeneralmente sí
El sistema puede funcionar sin internet para ventas, caja, inventario y reportes locales.

4. Impresora (opcional)

Para emitir tickets, se recomienda impresora térmica y driver Generic / Text Only o driver oficial del fabricante.

  • Mínimo: impresora térmica 58 mm.
  • Recomendado: impresora térmica 80 mm con driver oficial.

6. Instalación del sistema

  1. Ejecutar el archivo instalador .exe.
  2. Seguir pasos del asistente de instalación.
  3. Abrir el sistema y crear cuenta inicial.
  4. Configurar impresora si se usará ticket.
Si Windows muestra advertencia de seguridad, usar la opción Ejecutar de todas formas.

8. Recomendaciones finales

  • Usar una PC dedicada al sistema para evitar conflictos.
  • Respaldar información de forma periódica.
  • No apagar la PC de forma brusca durante operación.
  • Mantener usuarios y contraseñas bajo control del administrador.
Con estas condiciones, el sistema opera de forma estable en negocio real.
