Guia de Instalacion: Generic Text Only en Windows

Panel de Control - Sistema de Inventario y Ventas

📋 Pasos en esta guia

  • Abrir Configuracion de Windows
  • Ir a Impresoras y Escaneres
  • Agregar una impresora o escaner
  • Seleccionar impresora manualmente
  • Elegir el driver Generic / Text Only
  • Asignar nombre y finalizar
  • Establecer como impresora predeterminada
  • Probar impresion desde el sistema
1

Abrir Configuracion de Windows

Presione las teclas Windows + I al mismo tiempo para abrir la configuracion, o haga clic en Inicio y luego en el icono de engranaje.

🔍 Buscar en Windows
🔎 configuracion
⚙️ ConfiguracionAplicacion del sistema
💡 Tambien puede escribir "configuracion" en la barra de busqueda de Windows y hacer clic en el resultado.
2

Ir a Impresoras y Escaneres

Dentro de Configuracion, haga clic en Bluetooth y dispositivos (Windows 11) o Dispositivos (Windows 10), y luego en Impresoras y escaneres.

⚙️ Configuracion
🏠 Sistema
🖨️ Bluetooth y dispositivosImpresoras, escaneres, raton, teclado...
🌐 Red e Internet
↓
🖨️ Bluetooth y dispositivos

3

Agregar una impresora o escaner

En la pantalla de Impresoras y escaneres, haga clic en el boton Agregar dispositivo.

🖨️ Impresoras y escaneres
Buscar impresoras y escaneres...
+ Agregar dispositivo
🖨️ Microsoft Print to PDF
🖨️ Microsoft XPS Document Writer
💡 Si no encuentra su impresora, use la opcion "La impresora que quiero no esta en la lista".
4

Seleccionar impresora manualmente

Si Windows no detecta la impresora, haga clic en La impresora que quiero no esta en la lista.

Agregar impresora
Buscando impresoras...
No se encontraron impresoras
🔗 La impresora que quiero no esta en la lista
Agregar una impresora local o de red con configuracion manual
👆 Seleccione Agregar impresora local o de red con configuracion manual y haga clic en Siguiente.
5

Elegir el driver Generic / Text Only

Seleccione puerto USB001 (o el correcto) y en drivers use Generic como fabricante y Generic / Text Only como modelo.

Instalar el controlador de impresora
🏭 Epson
🏭 Fujitsu
🏭 Generic
🏭 HP
🖨️ Generic / Text Only
💡 Si no aparece Generic, haga clic en Windows Update y espere unos minutos.
6

Asignar nombre y finalizar

Escriba un nombre como Impresora Tickets y termine la instalacion.

Escriba un nombre de impresora
Nombre de impresora:
Impresora Tickets
✅ Impresora instalada correctamente
7

Establecer como impresora predeterminada

Abra la impresora instalada y pulse Establecer como predeterminada.

🖨️ Impresoras y escaneres
🖨️ Impresora Tickets - Predeterminada
🖨️ Microsoft Print to PDF
⚠️ En Windows 11, desactive primero "Permitir que Windows administre mi impresora predeterminada".
8

Probar impresion desde el sistema

Abra el Panel de Control, haga una venta de prueba y pulse Imprimir Ticket.

✅ Si el ticket sale correctamente, la instalacion fue exitosa.
⚠️ Si no sale nada, revise cable USB, encendido de impresora y vuelva a intentar.

Problemas comunes

  • Sale texto ilegible o simbolos raros: verifique que el driver sea exactamente Generic / Text Only.
  • No imprime nada: revise cable USB y que la impresora este encendida.
  • Sale en blanco: imprima pagina de prueba desde propiedades de la impresora.
  • Error al imprimir desde el sistema: reinicie la app y vuelva a intentar.
